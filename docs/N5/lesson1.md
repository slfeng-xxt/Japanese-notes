## 1.肯定判断句

```tips
【名词】 + は +【名词】+ です

(...是...)
```

- わたしは学生です。

## 2.否定判断句

```tips
【名词】 + は +【名词】+ ではありません

【名词】 + は +【名词】+ じゃありません　（口语）

(...不是...)
```

## 3.疑问句

```tips
【名词】 + は +【名词】+ ですか

(...是...吗)
```

### 3.1肯定回答

- はい、そうです。
- はい、〜です。

### 3.2否定回答

- いいえ、違います
- そうではありません
- 〜ではありません

### 3.3 选择疑问句

```tips
【名词】 + は +【名词】+ ですか ，【名词】+ ですか

(...是...吗，..还是...吗)
```

## 4.助词

### 4.1 も 

```tips
【名词】 + も 

【名词】 + も ＋【名词】 + も 

(...也)
```

- 彼は学生です。彼女も学生です。
- 彼も彼女も学生です。

## 4.2 の

- 表属性·表从属机构国家

```tips
【名词】 + の ＋【名词】 

(...的...)
```

- 日本の首都は東京です。
- 彼の本。

## 9.名字·名前·氏名

- 名字：姓，家族中国内地姓氏，日本姓氏
- 名前：名，个人的名字
- 氏名：全名

> 注意：日语「名字」≠ 中文「名字」，属于**中日同形异义词**。

| 日文 | 读音 | 含义 | 说明 |
|------|------|------|------|
| 名字 | みょうじ | 姓氏 | 专指**姓**，如「佐藤」「鈴木」 |
| 名前 | なまえ | 名字 / 姓名 | 最常用，可指全名也可单指名 |
| 氏名 | しめい | 全名 | 正式用词，填写表格时使用 |

**一句话记忆**：中文问「你叫什么名字？」→ 日语说「お名前は？」

例如：「田中太郎」

| 写法 | 读音 | 分类 |
|------|------|------|
| 田中 | たなか | 名字（姓） |
| 太郎 | たろう | 名前（名） |
| 田中太郎 | たなか たろう | 氏名（全名） |

> 「田中」是日本最常见的姓氏之一；「太郎」相当于中文的「小明」，是经典男性名（长男常用）。日本姓在前、名在后，与中文习惯一致。

## 10.年龄

| 日文 | 中文 | 使用场景 | 例句 |
|------|------|----------|------|
| 何歳（なんさい） | 几岁 | 标准问法，偏直接 | 何歳ですか？ |
| おいくつ | 几岁 | 礼貌问法，日常最常用 | おいくつですか？ |
| おとし（お歳/お年） | 年龄/年岁 | 礼貌名词，表「年龄」 | おとしは？ |

> **注意**：「おとずれ」意为「访れ（拜访）」，非年龄表达。表年龄应为 **おとし**。

## 11.「何」的两种读音

「何」有 **なに** 和 **なん** 两种读法，取决于后续接什么：

| 读音 | 后续接法 | 例词 | 中文 |
|------|----------|------|------|
| **なに** | ＋ 助词（が/を/に/も/か） | なにが | 什么（主格） |
| | ＋ で（手段） | なにで | 用什么 |
| | 单独使用 | なに？ | 什么？ |
| **なん** | ＋ です/だ | なんだ | 是什么 |
| | ＋ の | なんの | 什么的 |
| | ＋ 量词/数词 | なんさい | 几岁 |
| | ＋ で（原因） | なんで | 为什么 |
| | ＋ と | なんと | 多么 |
| | ＋ た/だ/な行音 | なんなの | 什么意思啊 |

> **核心规律**：「何」后接**助词**读 なに，后接**名词/です/量词**读 なん。
> **易混点**：「何で」读 なにで = 用何手段；读 なんで = 为什么。
