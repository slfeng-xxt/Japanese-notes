# 日本文构成

## 1.汉字

### 1.1字形

- 简体：先生・学生・政治
- 繁体：感動
- 有差别别：歩く
- 和制汉字：働く・気

### 1.2字义

- 与中文相同：学校・光明・国家
- 有差别：妻子（2个人）
- 完全不一致的：娘（女儿）・人参（胡萝卜）

### 1.3字音

- 音读：山（san）
- 训读: 山（ヤマ）

## 4.声调

- 声调： 头高型·中高型·尾高型·平板型（针对单词，所有单词只能有一处降调，接助词时，平板型要拉平，尾高型要下降）
- 表示方式：数字·横线
- 语调：平调·升调·降调。（针对句子）；平调-陈述句；升调-疑问句；降调-命令句/认可接受

### 4.1 声调类型图解（○低音 ●高音 以四拍词为例）

- **0型（平板型）**：第一拍是`低音`，从第二拍开始都是`高音`，后续助词也为`高音`

```
拍:  ①   ②   ③   ④  (助词)
     ○ → ● → ● → ● → ●
     低   高   高   高   高
```

- **1型（头高型）**：第一拍是`高音`，从第二拍开始都是`低音`

```
拍:  ①   ②   ③   ④
     ● → ○ → ○ → ○
     高   低   低   低
```

- **2型（中高型）**：第一拍是`低音`，第二拍是`高音`，从第三拍开始都是`低音`

```
拍:  ①   ②   ③   ④
     ○ → ● → ○ → ○
     低   高   低   低
```

- **3型（中高型）**：第一拍是`低音`，第二、三拍是`高音`，从第四拍开始`低音`

```
拍:  ①   ②   ③   ④
     ○ → ● → ● → ○
     低   高   高   低
```

- **4型（尾高型）**：第一拍是`低音`，第二拍到最后都是`高音`，但后续助词变为`低音`

```
拍:  ①   ②   ③   ④  (助词)
     ○ → ● → ● → ● → ○
     低   高   高   高   低
```

> **规律**：数字 N 表示「高音持续到第 N 拍为止」，第 N+1 拍开始降为低音。0型特殊，永不下降。
> **0型 vs 尾高型**：单独发音完全相同（都是 ○●●●），区别在于接助词时——0型助词保持高音，尾高型助词降为低音。

## 6.长音（長音）

日语中一个假名拖长一拍即为长音，规则如下：

| 假名段 | 长音标记 | 例词 | 读音 | 中文 |
|--------|----------|------|------|------|
| あ段 | + あ | おかあさん | okaasan | 妈妈 |
| い段 | + い | おいしい | oishii | 好吃 |
| う段 | + う | すうがく | suugaku | 数学 |
| え段 | + い / え | とけい | tokee | 时钟 |
| え段 | + い / え | おねえさん | oneesan | 姐姐 |
| お段 | + う / お | こうえん | kouen | 公园 |
| お段 | + う / お | とおい | tooi | 远的 |

> **规律**：各段假名 + 本段元音假名 = 长音。**例外**：え段多接「い」，お段多接「う」。

### 6.1 片假名长音

片假名长音统一使用「**ー**」（長音符），不区分段：

| 例词 | 读音 | 中文 |
|------|------|------|
| コーヒー | kōhī | 咖啡 |
| スーパー | sūpā | 超市 |
| テーブル | tēburu | 桌子 |
| ノート | nōto | 笔记本 |
| カード | kādo | 卡片 |
| パーティー | pātī | 聚会 |

> 与平假名不同，片假名**不论哪一段**，长音一律用「ー」表示，简单直接。

## 7.促音（そくおん）

促音用小写「**っ**」（平假名）或「**ッ**」（片假名）表示，本身不发音，而是**停顿一拍**，并强化后一个辅音。

### 7.1 出现位置

促音只出现在 **か・さ・た・ぱ** 四行假名之前：

| 类别 | 前行 | 例词 | 读音 | 中文 |
|------|------|------|------|------|
| か行前 | かきくけこ | がっこう | gakkō | 学校 |
| さ行前 | さしすせそ | ざっし | zasshi | 杂志 |
| た行前 | たちつてと | きって | kitte | 邮票 |
| ぱ行前 | ぱぴぷぺぽ | いっぱい | ippai | 满满 |

### 7.2 核心要点

| 要点 | 说明 |
|------|------|
| 占一拍 | 「っ」本身算一个音拍，要停顿 |
| 罗马字 | 后接辅音双写，如 がっこう → ga**kk**ō |
| 片假名 | 用「ッ」，如 ベッド (beddo) |
| 不可接 | 不出现在 な・ま・や・ら・わ 行及濁音前 |

> **对比**：「っ」= 停顿一拍，「ー」= 拉长一拍，两者易混淆，注意区分。

## 8.家族关系图谱

日语称呼自家和称呼别人家**用词不同**，体现内外有别：

```
        祖父（そふ） ─── 祖母（そぼ）
                    │
                父（ちち） ─── 母（はは）
                        │
            ┌─────────┼──────┬──────┐────────┐
            兄      姉       私     弟      妹
          （あに）  （あね） （わたし）（おとうと）（いもうと）
```

| 关系 | 自家（谦称） | 别人家（敬称） |
|------|-------------|----------------|
| 爸爸 | ちち（父） | おとうさん（お父さん） |
| 妈妈 | はは（母） | おかあさん（お母さん） |
| 哥哥 | あに（兄） | おにいさん（お兄さん） |
| 姐姐 | あね（姉） | おねえさん（お姉さん） |
| 弟弟 | おとうと（弟） | おとうとさん（弟さん） |
| 妹妹 | いもうと（妹） | いもうとさん（妹さん） |
| 爷爷 | そふ（祖父） | おじいさん（お爺さん） |
| 奶奶 | そぼ（祖母） | おばあさん（お婆さん） |
| 丈夫 | おっと（夫） | ごしゅじん（ご主人） |
| 妻子 | つま（妻） | おくさん（奥さん） |
| 儿子 | むすこ（息子） | むすこさん（息子さん） |
| 女儿 | むすめ（娘） | おじょうさん（お嬢さん） |

> **核心规则**：说自家用短读（ちち/はは），说别人家加「お」「ご」「～さん」。

### 8.1 兄弟姐妹总称

| 中文 | 日文 | 读音 | 说明 |
|------|------|------|------|
| 兄弟 | 兄弟 | きょうだい | 统称兄弟，也可泛指兄弟姐妹 |
| 姐妹 | 姉妹 | しまい | 专指姐妹 |

### 8.2 堂/表兄弟姐妹

日语不分「堂」和「表」，统一用「**いとこ**」：

> 日语中「いとこ」不区分堂/表、不分男女、不分长幼。如需区分，可说「男のいとこ」「年上のいとこ」等。


## 9.星期（曜日）

日语星期源于中国的**七曜**（日月火水木金土），规律统一：「～曜日（ようび）」：

| 星期 | 日文 | 读音 | 七曜 |
|------|------|------|------|
| 星期日 | 日曜日 | にちようび | 日（太阳） |
| 星期一 | 月曜日 | げつようび | 月（月亮） |
| 星期二 | 火曜日 | かようび | 火（火星） |
| 星期三 | 水曜日 | すいようび | 水（水星） |
| 星期四 | 木曜日 | もくようび | 木（木星） |
| 星期五 | 金曜日 | きんようび | 金（金星） |
| 星期六 | 土曜日 | どようび | 土（土星） |

> **记忆口诀**：日月火水木金土 + 曜日。只有「日曜日」读「にち」，其余曜读法各不同，需逐个记忆。
